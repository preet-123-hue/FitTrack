# App Icon Instructions

## Create Your App Icon

1. **Design Requirements:**
   - Size: 1024x1024 pixels (minimum)
   - Format: PNG with transparent background
   - Style: Simple, recognizable fitness-related icon
   - Colors: Use your app's brand colors

2. **Suggested Design:**
   - Walking person icon with step counter
   - Circular background with gradient
   - Clean, modern design

3. **Save the icon as:**
   - `app_icon.png` in this folder

4. **Generate icons:**
   ```bash
   flutter pub get
   flutter pub run flutter_launcher_icons:main
   ```
